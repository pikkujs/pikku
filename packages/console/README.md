# @pikku/console

The Pikku Console is a Vite + React SPA that provides a visual UI for managing Pikku projects, including workflow building, function inspection, and wiring configuration.

## Getting Started

```bash
yarn dev
```

Opens on [http://localhost:7070](http://localhost:7070).

## Build

```bash
yarn build
```

## Type Check

```bash
yarn lint
```
